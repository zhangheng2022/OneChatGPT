## 简介

OneChatGPT 是一个基于 Flutter 开发的开源聊天应用，支持多种 AI 模型，包括 GPT-3.5 和 Google Gemini 等。该项目旨在提供一个免费、易用的 AI 聊天平台。

## 功能特点

- 💬 支持多种 AI 模型对话
- 🔒 安全的用户认证系统
- 🌓 支持深色/浅色主题切换
- 💾 本地对话历史保存
- 📱 响应式界面设计

## 安装

目前该应用仅支持 Android 平台。您可以通过以下方式安装：

1. 从 Release 页面下载最新的 APK 文件
2. 在 Android 设备上安装该 APK

## 使用

1. 注册/登录您的账号
2. 选择想要使用的 AI 模型
3. 开始对话
